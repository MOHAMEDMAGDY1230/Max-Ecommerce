# 🛒 Max E-Commerce

A full-stack E-Commerce web application developed using **PHP**, **Laravel**, and **MySQL**. The system provides secure user authentication, product management, shopping features, and an online payment interface.

---

## 🚀 Features

- User Registration & Login
- Secure Password Hashing
- Product Listing
- Product Categories
- Product Details
- Shopping Cart
- Checkout System
- Payment Information Form
- Admin Product Management
- Responsive User Interface
- Input Validation
- Database Integration

## ⚙ Installation

1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/Max-Ecommerce.git
```

2. Open the project folder

```bash
cd Max-Ecommerce
```

3. Create a MySQL database

```
max_ecommerce
```

4. Import the SQL file into phpMyAdmin.

5. Update your database configuration.

6. Run the project using XAMPP or Laravel.
